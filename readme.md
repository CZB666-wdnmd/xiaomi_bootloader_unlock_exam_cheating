# 小米高考作弊


## 文件说明

- **question_db.txt**  
    放置题库数据。

- **rules.txt**  
    存放当天规则。程序启动后，会自动截屏识别，无需手动生成。

- **config.json**  
    用于存储通义千问和 Deepseek 的 API 配置。

## 配置示例

```json
{
        "ALIBABA_CLOUD_ACCESS_KEY_ID": "",
        "ALIBABA_CLOUD_ACCESS_KEY_SECRET": "",
        "DEEPSEEK_API_KEY": "",
        "DASHSCOPE_API_KEY": ""
}
```
